# Data Mining Project (Associations, Clustering, Supervised Learning)

This project applies multiple data mining techniques to analyze different datasets using IBM SPSS Modeler.

## Objectives
- Discover association rules in transactional data
- Perform clustering on the Iris dataset
- Build predictive models for housing prices

## Techniques Used
- Association Rule Mining
- Clustering (Unsupervised Learning)
- Supervised Learning Models
- Data preprocessing and normalization
- Training/testing partitioning

## Datasets
- Associations dataset
- Iris dataset
- Boston Housing dataset

## Key Takeaways
- Identified meaningful patterns through association rules
- Evaluated clustering performance across different cluster sizes
- Built predictive models using structured data

## Tools Used
- IBM SPSS Modeler
- Excel datasets
- Data mining workflows

## Why This Project Matters
This project demonstrates a broad understanding of core data mining techniques and the ability to apply different models depending on the problem type.
